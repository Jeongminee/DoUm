Hello, welcome to

DoUm

DoUm is a tutorial service for seniors
who have difficulty using smartphones.

The young generation were able to help the seniors, but our group thought of another way to resolve the problems seniors might face;
a solution that users can access easily, anytime.

Supporting Features

Tutorial

Encountered a problem while using an application? Go to the tutorial to find the solution to your problem right away

Tutorials provide...

- A step by step solution for the corresponding problem
- Screenshots with red circles that show you where to exactly tap on
- Short text descriptions that supports a clear guide

Course

Found a new application you want to learn but don't know where to start? Don't worry, we have the courses that allows you to completely master all the features of the application

Courses provide...

- A full course on all the features of the application
- Videos that show the exact screen of the application you want to learn
- An audio guide in the video that helps you learn the application not only visually but also auditory
